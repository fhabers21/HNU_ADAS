Simple Android App for the course "Advanced Driver Assistant Systems & Car Entertainment" at the University of Applied Sciences Neu-Ulm.
It's a basic app with two views. The User can enter a feedback. This feedback then will be sent via an API (PHP) to a Server (mysql).
The Server then checks if this feedback was already submitted and based on the outcome, the user gets a different answer displayed on the second screen in the App.

The Server used for this app are now offline.
